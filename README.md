## Overview

This project explores e-commerce performance using the Olist dataset, focusing on total realized revenue, operations overview, customer behavior, and data modeling.

- Built in Power BI
- Star schema data model
- ETL performed in Power Query

---

## Key Insights

- Total Realized Revenue: 15.42M (based on delivered orders)
- Average Delivery Time: 12.50 days
- Average Review Score: 4.09 / 5
